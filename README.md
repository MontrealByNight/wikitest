# Welcome to Montreal by Night!

This is a VTM RPG server. We use the rules and the system of VTM v5, with some slight adjustments and houserules to better fit the Play-by-Post format and the chronicle.

The game will touch dark subjects, violence, content warnings and so on. There could be graphic description of any of those subjects.
